# ObesityCare AI

Live Website:
https://obesity-care-ai.vercel.app

Backend API:
https://obesitycare-ai.onrender.com

Description :

ObesityCare AI is a full-stack web application that predicts obesity risk 
based on user lifestyle inputs such as weight, height, activity level, 
diet quality, sleep, and water intake.

The system calculates BMI and predicts the risk level as:

Low Risk
Average Risk
High Risk

Based on the prediction, the application also recommends a suitable diet plan.
